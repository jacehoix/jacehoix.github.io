业务分部深度解析

☁️ AI Cloud Infra（核心增长引擎）

  • Q4: $0.83B, 订阅收入YoY +143%
  • FY25: $2.83B, YoY +34%
  • 差异化: 全栈端到端能力（芯片昆仑芯→框架PaddlePaddle→ERNIE大模型→应用）
  • 企业认可度持续提升；大型客户通用AI基础设施使用增长
  • 昆仑芯拆分后独立上市将释放估值

🤖 AI-native 营销服务（爆发式增长）

  • Q4: $0.39B, YoY +110%
  • FY25: $1.40B, YoY +301%
  • 从传统竞价广告转向AI生成原生营销
  • ERNIE驱动的智能广告比传统效果更好
  • ⚠️ 高基数下2026年增速将显著放缓

📱 AI 应用（温和增长）

  • Q4: $0.39B; FY25: $1.46B, YoY +5%
  • ERNIE助手MAU: 2.02亿 (2025.12)
  • 百度App MAU: 6.79亿（持平）
  • Miaoda（AI编程）: IDC评为中国领先低代码生成式AI平台
  • ⚠️ 用户增长停滞，变现路径仍不清晰

🚗 Apollo Go 自动驾驶

  • Q4: 340万次完全无人驾驶, 峰值周>30万次, YoY +200%
  • 累计突破2000万次 (至2026年2月)
  • 全球26城市, 3亿+自动驾驶公里
  • 国际: 伦敦(Uber+Lyft)、阿布扎比、迪拜、首尔、香港
  • ⚠️ 仍持续烧钱，未见盈利时间表

📉 传统业务（Legacy Business，萎缩中）

  • Q4: RMB 12.3B (占百度GB的47%)
  • 传统搜索/信息流广告持续下滑
  • 被AI原生营销逐步蚕食但速度较慢
  • 仍是集团现金奶牛，利润率显著高于AI业务
  • 2026年预计继续下滑约5-10%

🎬 爱奇艺（稳定但不亮眼）

  • Q4: $0.97B (YoY +3%); FY25: $3.90B (YoY -7%)
  • Q4 Non-GAAP经营利润率 2%（全年2%）
  • 会员服务+内容授权是主要收入
  • 长期战略价值不明, 但拖累集团利润率
  • 是否分拆或出售? 管理层未有明确表示

📌 CEO 李彦宏开场 — 战略定调

「2025年是AI成为百度新核心的关键之年。AI Cloud Infra强劲增长，我们差异化的全栈端到端AI能力赢得了越来越多企业客户的认可。Apollo Go进一步巩固了全球领导地位，在行业领先规模下运营，同时加速国际市场拓展。AI原生营销服务持续增长，为长期发展开创了新可能。」
🟢 利好
AI战略定调清晰，全栈布局（芯片→模型→应用）是真正的技术护城河。AI原生营销+300%增速证明商业化落地有效。
「随着AI-first战略成形，我们对在AI时代创造持久价值充满信心。」
📊 中性
信心充足，但"持久价值"缺乏具体数字支撑。竞争对手（DeepSeek、字节、腾讯）均在加速追赶，差距正在缩小。

📌 CFO 何海剑 — 财务数据解读

「本季度，百度Core AI-powered业务营收超过110亿元人民币，占百度通用业务营收的43%。百度通用业务总营收环比增长6%，Non-GAAP经营利润环比增长28%至28亿元人民币。」
🟢 利好
AI业务占比43%是重要里程碑，意味着不到2年内将超过传统业务（Legacy Q4仍占47%）。利润率Q4回升到11%说明AI业务已有一定盈利能力。
「2025年下半年百度经营现金流转正，合计产生39亿元人民币，而上半年为负现金流。」
🟢 利好
H2现金流转正是关键信号，说明AI相关Capex投入在形成收入。但全年OCF仍为-30亿元，FCF-151亿元——主要因上半年大规模基础设施投入。
「董事会批准新股份回购计划，授权最高50亿美元，有效期至2028年12月31日。同时首次批准公司股息政策，首次派息可能于2026年底前执行。」
🟢 利好
$5B回购授权 + 首次股息政策是强烈的股东回报信号。以$41.6B市值计算，$5B回购相当于12%的股份。但执行力度和时间节点仍有不确定性。
「昆仑芯分拆及独立上市正在推进中，我们相信这将为股东释放重大价值。」
🟢 利好
昆仑芯是中国领先AI芯片厂商，独立上市后可能获得远高于百度整体的估值倍数。但上市时间和发行方式仍不确定。
「全年营收同比下降3%至1290亿元，主要由传统业务（Legacy Business）下滑所致，部分被AI-powered业务的增长抵消。」
🔴 利空
全年营收-3%是百度连续第二年下滑。传统搜索广告的失血速度仍快于AI业务的输血速度。若2026年趋势不扭转，市场信心将进一步受损。
「核心资产组商誉及长期资产减值16.2亿元人民币，导致全年GAAP经营亏损58亿元人民币。」
🔴 利空
162亿RMB减值是重大警示——意味着之前的投资（研发、并购、基础设施）未能产生预期回报。管理层没有明确说明减值具体对象，透明度不足。

📌 Q&A — AI Cloud 增长可持续性

AI Cloud Infra订阅收入Q4同比增长143%——分析师询问：这种增速能否持续？是否有大客户集中风险？
📊 中性
143%增速很亮眼，但基数效应明显。预计2026年增速将放缓至40-60%区间。大客户集中风险是隐患，管理层未披露前五大客户占比。
管理层强调百度的差异化在于「全栈能力」：昆仑芯AI芯片 → PaddlePaddle框架 → ERNIE基础模型 → 企业AI应用，形成闭环。竞争对手难以复制整条链路。
🟢 利好
全栈布局是真实的护城河，降低了对NVIDIA等海外供应商的依赖。在中国AI国产化背景下，昆仑芯有政策红利。但与华为昇腾、寒武纪等国产芯片的竞争同样激烈。

📌 Q&A — Apollo Go 商业化路径

分析师询问：Apollo Go 单量同比+200%，但何时实现收支平衡？自动驾驶投资回报时间线？
📊 中性偏空
管理层回避了具体盈利时间表，只强调"规模化是关键"。以目前26城市运营看，单车经济性改善有限，预计2027-2028年前仍持续亏损。
「Apollo Go在阿布扎比、迪拜、伦敦等市场的国际扩张，证明我们的技术在全球范围内具有竞争力和可部署性。与Uber、Lyft等平台合作进一步扩大了我们的触达范围。」
🟢 利好
国际市场拓展是百度少有的正面叙事。与Uber/Lyft合作是轻资产扩张策略，避免了在海外自建运营的高成本。若欧美市场准入打通，估值想象空间巨大。

📌 Q&A — 资本配置与股东回报

分析师询问：FY2025 Capex同比增长48%至170亿元，2026年Capex指引如何？FCF何时转正？
🔴 利空
管理层未给出明确Capex指引，只说"继续投资AI基础设施"。若2026年Capex维持高水平，FCF依然难以转正。投资者需要更明确的资本支出纪律信号。
「$5B回购计划的规模反映了我们对公司内在价值的信心和对股东回报的承诺。首次股息政策标志着百度迈入新阶段——成熟稳健的现金流生成者。」
🟢 利好
以市值$41.6B计算，$5B回购是重量级承诺（12%）。但注意：当前FCF为负，回购资金需动用现金储备（$42B总现金中大量为受限或长期）。首次分红更多是姿态性信号。

📌 Q&A — 传统业务萎缩速度

分析师询问：传统搜索广告（Legacy Business）Q4 RMB 12.3B，下滑趋势是否加速？AI搜索是否在蚕食自身业务？
🔴 利空
这是百度面临的"创新者困境"——ERNIE加持的AI搜索正在替代传统付费竞价排名，而AI营销的单次价值高于传统广告但总量尚不足以弥补缺口。2026年Legacy下滑可能加速。
管理层表示AI-native营销服务正逐步将传统广告主迁移到新模式，「这是主动的业务演进，而非被动的流失」。AI营销的CPM和转化率均优于传统广告。
📊 中性
主动迁移vs被动流失的叙事逻辑上成立，但实际上广告主同时在投其他平台（抖音、微信等）。百度搜索的市场份额从高峰的75%已降至约50%。

📌 Q&A — ERNIE 5.0 与竞争格局

「2026年1月发布的ERNIE 5.0是原生全模态基础模型，在多项基准测试中表现领先。我们将持续迭代，保持领先优势。」
📊 中性
ERNIE 5.0发布时间恰在DeepSeek R1引发全球轰动之后，有跟风应对之嫌。评测基准领先并不等于实际应用领先，市场口碑仍待验证。
分析师询问：DeepSeek的崛起对百度AI Cloud业务影响如何？开源模型是否压制了企业对百度私有云的需求？
🔴 利空
DeepSeek是百度最大的短期威胁——高性能开源模型降低了企业使用AI云的门槛和成本，直接冲击百度Cloud的定价权。管理层没有正面回答这个问题。

巴菲特六维评分
B+

护城河 — 技术壁垒存在，但正在被侵蚀

全栈AI能力（昆仑芯+PaddlePaddle+ERNIE+应用）是真实护城河，非硬件公司难以复制。Apollo Go的300M自动驾驶公里数据是竞争壁垒。但核心搜索的市场份额从75%降至~50%，ERNIE面临DeepSeek、字节豆包、腾讯混元的激烈竞争，护城河正在收窄。

B

管理层 — 技术远见强，资本纪律待验证

李彦宏的AI战略布局（2012年就成立深度学习研究院）证明了前瞻性。$5B回购+首次分红是股东友好信号。但162亿减值暗示之前有错误投资，Capex从81亿暴增到120亿且无指引，让人担心资本配置纪律。管理层电话会议的问题回避策略（DeepSeek、减值细节）透明度不足。

B+

财务健康 — 现金堡垒极其强大

总现金及投资$42.06B ≈ 市值$41.6B，极端情况下"零价格买业务"。净债务为负，利息收入$1.23B/年（RMB 86亿）。但FY25 OCF -$0.43B、FCF -$2.16B需要关注。应收账款同比增加28%（12.97B vs 10.10B），可能是信用风险信号，也是SGA增加的主因（预期信用损失）。

B-

盈利质量 — 结构性压力明显

Non-GAAP净利率从2024年20%降至2025年15%，趋势向下。AI Cloud毛利率推测低于传统广告（服务器成本高），结构性压力将在2026年持续。一次性减值162亿后，利润质量存疑——管理层没有解释具体减值对象。R&D费用下降8%意味着削减了人员，短期提升利润率但可能损害长期竞争力。

C+

可预测性 — 多重不确定性

AI Cloud增速受DeepSeek冲击、Legacy下滑速度、Apollo烧钱节奏、中美科技博弈均高度不确定。2025连续两年营收下滑打破了百度的"稳定性"标签。用户数持平、市场份额继续下滑意味着平台量没有增长。巴菲特最讨厌的转型期不确定性，百度目前正处其中。

B-

资本纪律 — 回购是好信号，但Capex令人不安

$5B回购 + 首次分红 = 积极信号。但FY25 Capex 120亿（同比+48%）且无明确指引，在FCF为负的情况下大手笔投资AI基础设施，是典型的"科技公司先花后赚"模式。如果2026年AI Cloud收入增速不能超越Capex增速，这笔投资将进一步损害股东价值。

🍔 巴菲特风格深度点评

百度是我见过的最奇特的投资标的之一。市值$416亿，现金及投资$421亿——你用零价格就能买到这整个企业，然后连带赠送一家全球最大的中文AI公司、一个正在国际扩张的自动驾驶领导者，以及一个快速增长的AI云业务。

听起来很完美，对吧？且慢。

我做生意70年，学到最重要的一课是："看起来便宜"和"真的便宜"是两回事。一家公司手握$421亿现金，代表管理层有机会创造巨大价值，或者毁掉巨大价值。两种结果都发生过。

百度让我担心的不是AI技术——技术上他们确实领先，全栈布局是真实的护城河。让我担心的是：FY2025全年经营现金流为负$4.3亿，自由现金流负$21.6亿，同时Capex同比暴增48%，且管理层拒绝给出2026年的资本支出指引。这不是一家"保守稳健"管理的公司应有的样子。

更令人困惑的是那笔162亿人民币的减值。管理层说是"核心资产组"，但什么是核心资产组？哪些资产从高价买入、现在不值钱了？我问过投行分析师，没人说得清楚。不透明的减值是危险信号——它意味着之前的投资决策出了问题，而你还不知道问题有多深。

好消息是：$5B回购授权加上首次分红政策，说明管理层终于意识到手里的现金应该部分回馈股东。这个信号比任何季度财报都重要——它意味着百度从"科技成长股"叙事转向"价值+成长"叙事。

关于Apollo Go：我不投自己看不懂的业务。一辆无人驾驶汽车在阿布扎比和伦敦行驶的商业模式，我没有足够的理解框架来判断它的价值。我知道的是：它每年烧很多钱，而"何时盈利"这个问题没有人能回答。

我的结论：百度的安全边际来自现金，而不是业务。在AI转型完成之前，这家公司是一个"期权"——如果AI战略成功，现在的价格是便宜的；如果失败，现金保护了你的下行空间。但期权不是我喜欢的投资方式。我喜欢护城河清晰、现金流稳定、管理层透明的企业。

如果你一定要买，等到$90-100美元——那时候你是在用九折价格买净现金，业务完全免费。那才是真正的安全边际。现在的$119，我宁愿观望。
