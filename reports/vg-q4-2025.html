公司概况 — 美国LNG出口新巨头
Venture Global 是美国快速崛起的LNG（液化天然气）生产商和出口商，总部位于弗吉尼亚州阿灵顿。公司在路易斯安那州墨西哥湾沿岸开发、建设和运营大型LNG液化和出口设施。

三大核心资产：
• Calcasieu Pass (CP) — 已商业运营 (2025年4月COD)，约10 MTPA产能
• Plaquemines LNG — 调试阶段，36台液化列车全部启动，2026年Q4目标COD，约20 MTPA
• CP2 LNG — 在建，Phase I预计2027年底首产，Phase II 2026年3月获FID，约29 MTPA

三大LNG项目详解

🟢 Calcasieu Pass (CP) — 已商业运营

  • 状态：2025年4月达到COD（商业运营日），美国第7大LNG出口设施
  • 产能：约10 MTPA（peak），18台液化列车
  • Q4出口：38船（略低预期，受船舶可用性+大西洋风暴影响）
  • 液化费：$2.01/MMBtu（Q4实现，含仲裁准备金）；2026E $1.98/MMBtu
  • 仲裁：Repsol案1月获胜（无赔偿责任）；BP案进行中，估计对收入影响约$1,300万/季
  • 盈利能力：稳定运营中，但受长期SPA定价限制，利润率低于调试期Plaquemines

📊 Plaquemines LNG — 调试冲刺阶段

  • 状态：全部36台液化列车已初始启动，日产超1船调试货物
  • 产能：约20 MTPA（peak）；已申请FERC扩容至35 MTPA
  • 2025出口：234船（占全公司62%）
  • 液化费：$6.02/MMBtu（Q4调试期费率，远高于CP的$2.01）
  • 时间线：Q2 2026切换永久电力，夏季完成EPC实质完工，Q4 2026 目标COD
  • 融资：发行$30亿票据，偿还$32亿建设贷款
  • 扩张：已向DOE和FERC申请31 MTPA bolt-on扩建

🔵 CP2 LNG — 在建，规模最大

  • Phase I：按预算按时推进，首个LNG储罐创行业最快封顶记录
  • 26台液化列车中6台已交付上基础，预处理模块即将安装
  • Phase I 预计2027年底首产，产能约20 MTPA
  • Phase II：2026年3月获FID，追加约9 MTPA
  • 合计产能：29 MTPA（peak），几乎全部签约
  • 已投入$17亿权益资金，项目融资已完成
  • 计划bolt-on扩建 +6.4 MTPA

📈 商业签约 & 全球布局

  • 总签约：49 MTPA 长期/中期购销协议，$1,340亿+合同收入
  • 2025年新签：9.25 MTPA 二十年SPA
  • 新客户：韩华航空（1.5 MTPA / 20年），Trafigura（0.5 MTPA / 5年，净利差>$3）
  • 2026年已签约比例：69%的预期货物
  • 客户分布：欧洲、亚洲、全球贸易商，高度分散化
  • 核心优势：低成本、高产量、模块化建设——行业最快建设速度
电话会议完整内容 — 逐条记录 + 利好/利空判断

📌 CEO 开场 — Michael Sabel

「2025年是公司具有里程碑意义的一年。我们完成了1月的IPO、4月Calcasieu Pass商业运营、Plaquemines持续调试斜坡式爬产，以及CP2启动建设并完成Phase I融资。我们正在同时建设57 MTPA以上的产能。」
🟢 利好
同时推进两个超大型项目的执行力令人印象深刻。57 MTPA在建产能相当于Cheniere Energy全部运营产能，体现了极强的项目管理能力。
「Plaquemines现在每天产出超过一船调试货物。所有36台液化列车已完成初始启动。我们预计Phase I将在Q2切换到永久电厂配置，在夏末实现EPC实质完工。」
🟢 利好
日产一船是Plaquemines产能爬升的重要里程碑。36台列车全部启动意味着产能验证基本完成，2026年COD目标可信度大幅提升。
「CP2 Phase I建设按预算按时推进。首个LNG储罐以行业创纪录的速度完成封顶。6台液化列车已交付并安装到基础上，首个预处理模块即将到位。」
🟢 利好
CP2建设提前于进度是少见的好消息——LNG项目通常延期超支。创纪录的封顶速度验证了VG模块化建设方法的优势。
「自去年4月重新进入签约市场以来，我们签署了9.25 MTPA的新20年SPA。今天额外宣布与Trafigura签署0.5 MTPA的5年合同（净利差超过$3），以及与韩华航空签署1.5 MTPA的20年SPA——这是我们首个韩国长期客户。」
🟢 利好
Trafigura的$3+净利差体现了中短期合同的高利润率。韩国客户的突破意味着亚太市场开拓加速，且20年期限锁定了长期收入可见性。
「公司拥有超过$1,340亿的已签约第三方收入，以及49 MTPA的长期和中期购销协议基础。总资产在年内增长约$100亿至$534亿。」
🟢 利好
$1,340亿合同收入提供了极高的长期现金流可见性。对于一个市值$400亿的公司来说，合同价值是市值的3.3倍——未来收入的确定性非常高。
「Calcasieu Pass在Q4出口38船，略低于预期——主要因船舶可用性和大西洋风暴延误。Repsol仲裁案我们在1月获得'无赔偿责任'的有利裁决。对于其余仲裁案（含BP），目前最佳估计是每季度减少CP收入约$1,300万的非现金准备金，贯穿20年SPA期限。」
📊 中性
Repsol案胜诉是好消息，但BP仲裁持续是不确定因素。$1,300万/季的影响虽小（不到收入的1%），但20年期限意味着累计影响约$10.4亿——不算无足轻重。
「我们已向FERC申请将Plaquemines和CP2的授权峰值液化产能提高至35 MTPA，并向FERC和DOE申请在Plaquemines进行最多31 MTPA的bolt-on扩建。我们预计将在CP2和Plaquemines都进行bolt-on扩建——每个约增加6.4 MTPA。」
🟢 利好
Bolt-on扩建是最优的增长路径——利用现有基础设施、约20个月建成、资本效率远高于绿地项目。如果获批，总产能可从目前的~60 MTPA提升至~73 MTPA。

📌 CFO 财务详述 — Jack Thayer

「Q4营收$44亿，同比增长192%，主要由更高的销售量驱动（478 TBtu vs 128 TBtu），部分被更低的净费率抵消。全年营收$138亿，同比增长177%。Q4调整后EBITDA $20亿，同比增长191%；全年调整后EBITDA $63亿，同比增长198%。」
🟢 利好
近3倍的营收增长和EBITDA增长令人震撼，主要得益于Plaquemines产量爬坡。值得注意的是EBITDA增速略低于营收增速，说明成本也在同步上升。
「Q4归母净利润$10.7亿，同比增长23%——增速远低于EBITDA的191%。这主要因为更高的利息支出和利率互换变动的负面影响。全年利息支出从$5.84亿增长至$14.54亿——增长了149%。」
🔴 利空
利息支出暴增149%是最大隐忧。$14.5亿的年度利息占EBITDA的23%，意味着每赚$4就有$1给了债主。高杠杆在增长期可接受，但利率风险和再融资风险不容忽视。
「融资方面：我们发行了$30亿Plaquemines票据并偿还了$32亿建设贷款，同时获得了$20亿公司循环信贷额度（年末未动用）。」
📊 中性偏好
债务结构优化（建设贷→票据）降低了短期再融资压力。$20亿未动用的循环额度提供了充足的流动性缓冲。
「Plaquemines Q4实现加权平均液化费$6.02/MMBtu，但受到12月Henry Hub价格上涨、航运日租金上升和TTF基本持平的压缩。Calcasieu Pass Q4实现$2.01/MMBtu，2026年预计$1.98/MMBtu。」
📊 中性
Plaquemines调试期的$6.02利润丰厚但不可持续——COD后将按SPA合同定价（通常$3-4）。CP的$2.01反映成熟资产的稳定但较低回报率。关键看合同组合的加权利差。

📌 2026年指引与市场展望

「2026年预计出口486-527船：Calcasieu Pass 145-156船，Plaquemines 341-371船。Plaquemines范围较宽反映调试不确定性和潜在修复中断。」
📊 中性
指引比2025年实际（380船）增长28-39%，增量全部来自Plaquemines继续爬坡。但宽幅指引（41船差距）暗示执行仍有不确定性。
「2026年调整后EBITDA指引$52-58亿，假设剩余未售货物的液化费$5-6/MMBtu。液化费每变动$1/MMBtu，EBITDA影响约$5.75-6.25亿。」
🔴 利空偏中性
2026E EBITDA指引中值$55亿 低于 2025A的$63亿——这意味着利润率压缩。核心原因是Plaquemines从高利润的调试期切换到正常SPA定价。市场看到的是"增速放缓"。
「Q1 2026将受到Winter Storm Fern的影响和利润率压缩。目前2026年69%的潜在货物已签约，预计签约比例还将随新合同签署继续提升。」
📊 中性
Q1季节性弱势+风暴影响已被提前沟通。69%签约率提供底部保障，但31%敞口暴露在现货价格波动下。
「CP2 Phase II FID预计在2026年上半年完成，目前已签署5 MTPA的20年SPA支持融资。我们已投入$17亿权益资金。」——注：Phase II于2026年3月14日正式获得FID和项目融资交割。
🟢 利好
Phase II FID已于3/14达成，是VG第五个FID——不到7年实现5个FID在LNG行业史无前例。这进一步验证了VG的融资能力和项目执行力。
「美国将在稳定全球LNG市场中发挥关键角色。VG拥有市场上最大的可用货物量之一。低且稳定的LNG价格长期可以增加需求。航运约束和费率是近期关键变量。」
📊 中性
管理层对LNG宏观前景乐观但也坦诚面临航运瓶颈。"最大可用货物量"说明VG已成为全球LNG定价的重要参与者。

巴菲特六维分析
B

护城河：规模+速度优势，但LNG是大宗商品

VG的核心竞争力是建设速度（行业最快）和低成本模块化方法。但LNG本质上是大宗商品——没有定价权，利润取决于Henry Hub与TTF/JKM的价差。长期SPA提供了收入确定性，但不等于护城河。真正的壁垒是$500亿+的资产规模和FERC审批门槛。

B+

管理层：极强的执行力，但杠杆使用激进

CEO Michael Sabel带领VG在不到7年内完成5个FID——LNG行业史无前例。项目按时按预算推进是少见的。但管理层对杠杆的使用极其激进（净债务/EBITDA 5.2x），如果项目延期，后果严重。IPO定价$23-27也显得过于自信。

C+

财务健康：盈利优秀但负债惊人

经营层面非常健康：$65.7亿经营现金流、37%经营利润率、$63亿EBITDA。但$349亿总债务是市值的近9倍（！），利息支出$14.5亿吞噬了EBITDA的23%。净债务$323亿意味着这是一个高度杠杆化的赌注——押注LNG需求持续增长+项目按时完工。

A-

盈利质量：合同收入高度可预测

$1,340亿已签约收入提供了极高的长期现金流可见性。49 MTPA的20年SPA意味着未来10-20年的收入基本锁定。这是巴菲特最喜欢的"收费桥梁"模式——建好桥，收20年过路费。但调试期利润率不可外推，COD后利润率将回归正常。

B-

可预测性：长期清晰，短期波动大

一旦所有项目投产（2028+），盈利可预测性极高——固定合同+低变动成本。但当前处于建设/调试阶段，季度业绩波动巨大（调试期vs运营期费率差异、天气影响、航运约束）。此外，BP仲裁和LNG价格波动是持续的不确定因素。

C

资本配置纪律：极度激进的扩张

VG的策略是"全力加速"——同时建设57 MTPA产能、每年$130亿+资本支出、净债务从$100亿升至$323亿。这不是巴菲特式的保守资本配置。IPO融资$18亿、公司股息仅$0.067/股，说明所有现金都投入了增长。如果LNG市场转向——供过于求或价格暴跌——这种激进扩张可能变成致命弱点。

Plaquemines日产一船LNG——这台机器正在加速。CP2创下行业最快封顶记录。管理层的执行力不是说说而已，是用钢筋水泥证明的。如果一切按计划走，到2028-2029年，这家公司将拥有$90-120亿的稳态EBITDA，以当前估值计算，稳态EV/EBITDA仅6-8倍——这在基础设施领域是非常便宜的。

然后是让我睡不着觉的部分。
